    How to check windows version:

    • Press Windows Key + R at the same time
    • In the run window (In the bottom left) type "winver"

    • The windows version should be located in the location of the highlighted part of the image above
    • In the instance of the image above the windows version is 21H2
    • The windows version is important as some products might not support your version (Please read product description for PC requirements such as windows version)
